# AI-Powered Study Assistant using Google Gemini

An intelligent study assistant built with Python and Streamlit, 
powered by Google Gemini AI.

## Features
- Smart Summarizer
- Concept Explainer  
- Interactive Quiz
- Active Flashcards

## Tech Stack
- Frontend: Streamlit
- Backend: Python
- AI: Google Gemini API
- Dataset: AI Flashcards 300K (Kaggle)

## How to Run
1. Install requirements: pip install -r requirements.txt
2. Run: streamlit run app.py
